Estadstica bayesiana con Stan - 01 Modelo-Gamma

En este notebook se presenta el uso de cmdstanpy para simular muestras de la distribucin predictiva de un modelo simple Normal-Gamma.

$$ y_i \sim Normal(\mu, \tau), i, i = 1\ldots, n, $$

$$ \mu \sim Normal(0, \lambda), $$

$$ \tau \sim Gamma(\alpha, \beta), $$

con $\lambda$, $\alpha$ y $\beta$ conocidos.

Definicin del modelo en Stan

El primer paso es definir el modelo en un archivo independiente y guardarlo en con extensin '.stan'.

El script contiene 3 partes: data, parameters y model.

data {
    int<lower=0> N;                // Nmero de observaciones
    array[N] real y;       // Datos
    real<lower=1e-3> lambda;       // Parmetro de la normal
    real<lower=1e-3> alpha;        // Parmetro de la gamma
    real<lower=1e-3> beta;         // Parmetro de la gamma
}
parameters {
    real mu;           // Parmetro latente
    real<lower=1e-3> tau;          // Parmetro latente
}
model {
    mu ~ normal(0, lambda);          // Prior
    tau ~ gamma(alpha, beta);      // Prior
    y ~ normal(mu, tau);           // Likelihood
}

Datos

Para este ejemplo, consideremos que el valor verdadero (?) de $\lambda$ es 5. Simulemos $N=150$ valores de una distribucin $Poisson(5)$.

En este ejemplo sencillo, vamos a declarar $\alpha$ y $\beta$ para la distriucin inicial (prior) de $\lambda$, de manera uqe sea poco informativa.

Al final creamos un diccionario que usaremos como input para ajustar el modelo con Stan.

In[2]:
np.random.seed(42)
N = 150  # Nmero de observaciones
mu = 15 # lambda verdadero
tau = 1.7
y_obs = np.random.normal(mu, tau, N)  # Datos Poisson

lambda_, alpha, beta = 10, 1, 0.01  # Parmetros de la distribucin gamma

# Input del modelo
data = {
    "N": N,
    "y": y_obs.tolist(),
    "lambda":lambda_,
    "alpha": alpha,
    "beta": beta
}

Compilacin del modelo y ajuste (simulacin)

Instanciamos el modelo proporcionando la ruta del archivo .stan.

In[3]:
model = CmdStanModel(stan_file="normal_gamma.stan")

fit = model.sample(
    data=data,
    iter_sampling=2000,
    iter_warmup=1000,
    chains=4,
    parallel_chains=4
)
